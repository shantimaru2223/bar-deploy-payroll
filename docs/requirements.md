# 要件定義 - Bar Deploy 給与明細アプリ

## 1. プロジェクト概要
- **店舗名**: Bar Deploy
- **目的**: バイトスタッフ向けの給与明細を簡単に作成・PDF出力できるWebアプリ
- **利用者**: 店舗オーナー（1名）
- **対象スタッフ数**: 1〜5名

## 2. 機能要件

### 2.1 スタッフ管理
- スタッフの登録（名前、給与タイプ（時給/月給）、給与額、交通費）
- スタッフ情報の編集・削除
- スタッフ一覧の表示

### 2.2 勤怠入力
- 対象月を選択して勤務データを入力
- 入力項目: 勤務日、勤務開始時間、勤務終了時間
- 勤務時間の自動計算（終了 - 開始）

### 2.3 給与計算
- **時給スタッフ**: 総勤務時間 × 時給 = 基本給
- **月給スタッフ**: 月給額がそのまま基本給となる
- 基本給 + 交通費 = 総支給額
- **源泉徴収税額**: 基本給 × 10.21%（端数切り捨て）
- 差引支給額 = 総支給額 - 源泉徴収税額
- 金額は全て整数（円単位）で計算する

### 2.4 給与明細の表示・出力
- 給与明細の画面表示
- PDF形式での出力・ダウンロード

### 2.5 明細に含める項目
| 項目 | 説明 |
|---|---|
| スタッフ名 | バイトの名前 |
| 対象期間 | 給与の対象年月 |
| 勤務日数 | その月の出勤日数 |
| 総勤務時間 | その月の合計勤務時間 |
| 給与タイプ | 時給 または 月給 |
| 時給 / 月給額 | 時給の場合は1時間あたり、月給の場合は月額 |
| 基本給 | 時給: 総勤務時間 × 時給 / 月給: 月給額 |
| 交通費 | 月額の交通費 |
| 総支給額 | 基本給 + 交通費 |
| 源泉徴収税額 | 基本給 × 10.21% |
| 差引支給額 | 総支給額 - 源泉徴収税額 |

## 3. 非機能要件
- ブラウザで動作すること（PC・スマホ対応）
- データはローカルまたはシンプルなDBに保存
- 操作が直感的でシンプルであること
- 個人情報を扱うため、最低限のセキュリティを確保する

## 4. 対象外（今回のスコープ外）
- 社会保険料の控除計算
- 深夜手当・残業代の自動計算
- 賞与の計算
- スタッフ自身がログインして閲覧する機能
