# 要件定義 - Bar Deploy 給与明細アプリ

## 1. プロジェクト概要
- **店舗名**: Bar Deploy
- **目的**: バイトスタッフ向けの給与明細を簡単に作成・PDF出力できるWebアプリ
- **利用者**: 店舗オーナー（1名）
- **対象スタッフ数**: 1〜5名

## 2. 機能要件

### 2.1 スタッフ管理
- スタッフの登録（名前、給与タイプ（時給/日給/月給）、給与額、ドリンク単価、交通費（片道））
- スタッフ情報の**編集**・削除
- スタッフ一覧の表示

### 2.2 勤怠入力
- 対象月を選択して入力
- **時給スタッフ**: 勤務日・開始・終了を入力（勤務時間を自動計算）。終了≦開始は「翌日終了」として +24時間で計算し、画面に明示する
- **日給スタッフ**: その月の出勤日数をまとめて入力
- **全タイプ共通**: その月のドリンク杯数を入力

### 2.3 給与計算
- **時給スタッフ**: 総勤務時間 × 時給 = 基本給（分単位で計算・端数切り捨て）
- **日給スタッフ**: 日当 × 出勤日数 = 基本給
- **月給スタッフ**: 月給額がそのまま基本給
- **ドリンクバック（全タイプ共通）**: ドリンク杯数 × ドリンク単価
- **交通費**: 片道運賃 × 2（往復）× 出勤日数
- 総支給額 = 基本給 + ドリンクバック + 交通費
- **源泉徴収税額**: (基本給 + ドリンクバック) × 10.21%（端数切り捨て、**交通費は対象外**）
- 差引支給額 = 総支給額 - 源泉徴収税額
- 金額は全て整数（円単位）で計算する

### 2.4 給与明細の表示・出力
- 給与明細の画面表示
- PDF形式での出力・ダウンロード

### 2.5 明細に含める項目
| 項目 | 説明 |
|---|---|
| スタッフ名 | バイトの名前 |
| 対象期間 | 給与の対象年月 |
| 給与タイプ | 時給 / 日給 / 月給 |
| 勤務日数 | その月の出勤日数 |
| 総勤務時間 | （時給）その月の合計勤務時間 |
| 時給 / 日当 / 月給額 | 給与タイプに応じた単価 |
| 基本給 | タイプ別に算出（上記 2.3） |
| ドリンクバック | 杯数 × 単価（0杯なら明細では非表示） |
| 交通費 | 片道 × 2 × 出勤日数 |
| 総支給額 | 基本給 + ドリンクバック + 交通費 |
| 源泉徴収税額 | (基本給 + ドリンクバック) × 10.21% |
| 差引支給額 | 総支給額 - 源泉徴収税額 |

## 3. 非機能要件
- ブラウザで動作すること（PC・スマホ対応）
- データは SQLite（ファイル1つ）に保存
- 操作が直感的でシンプルであること
- **入力値の検証**を行う（画面側で即時表示、サーバ側でも検証して不正を弾く）
- 個人情報を扱うため、最低限のセキュリティを確保する

## 4. 対象外（今回のスコープ外）
- ログイン認証（将来検討）
- 社会保険料の控除計算
- 深夜手当・残業代の自動計算
- 賞与の計算
- スタッフ自身がログインして閲覧する機能
